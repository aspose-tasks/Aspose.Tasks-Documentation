The [ProjectServerManager ](https://apireference.aspose.com/tasks/java/com.aspose.tasks/ProjectServerManager)class provides the methods to retrieve projects from the specified Project Online account. The ProjectServerCredentials class shall be used to provide credentials that are used to connect to Project Online. Aspose.Tasks API provides an option to specify SiteUrl, username, and password in order to create a connection to Project Online.
